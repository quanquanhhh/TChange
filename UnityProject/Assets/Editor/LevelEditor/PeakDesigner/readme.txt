Page1 - 定义单个山峰的布局模板（比如 3 层三角形）
功能点：
自定义高宽、层数、卡牌间隔
鼠标点击设置卡牌位（是否有牌、位置）
储存为 “PeakTemplate.asset” 文件（ScriptableObject）
支持导入/导出 JSON 或预览渲染


Page 2：关卡结构编辑器（Level Composer）
将多个“峰”组合成完整关卡结构
功能点：
拖拽多个 PeakTemplate 组成一个布局（可翻转/缩放）
设置全局参数：
目标类型（清空/得分）
最大步数 / combo 要求
drawPile 长度范围
难度标签（简单/中等/困难）
自动分配卡位 index
预览所有卡牌 slot 位置



Page 3：DrawPile / Card 放置页
以规则生成或手动填入 drawPile 和桌面卡牌
功能点：
按参数随机生成 drawPile
放置桌面卡（点数 / 是否可见）
可视化每张牌连接路径（辅助设计连击）
支持保存“随机种子”用于重现关卡



Page 4：解法验证器（Solver）
模拟打牌过程，验证该关卡是否“至少有解”
功能点：
内部使用 DFS / BFS / IDA* 模拟打牌过程
输出是否可解、combo最大值、步数区间
如不可解：可“跳转返回上一步”重新生成
可批量验证多个关卡（用于工具化产出）
